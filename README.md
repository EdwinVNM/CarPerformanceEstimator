# CarPerformanceEstimator
Accurate Sport Car Performance Estimator


An Accurate Sport Car Performance Estimator based on Generation, Model Variant, Power, Torque, Conditions (Elevation, Temperature) and fuel type (Unleaded: 91 OCTANE / 93 OCTANE / 100 OCTANE / E85). Once you input these variables the python program uses information about the car such as mass, frontal area, coefficient of drag (CD), drivetrain efficiency, redline RPM, gear limited vmax and coefficient of friction (μ) for braking to determine the acceleration, maximum velocity and braking times for the car. Currently my most accurate Model is the 992.1 Carrera 4S with dyno curves compared and based off of M Engineering. When I get the chance I will improve accuracy and add more Models based off of user feedback. But first I need to update the UI for optimal user experience. I have extensive experience with both Mobile Apps and Web Applications so expect to see consumer oriented features soon.
